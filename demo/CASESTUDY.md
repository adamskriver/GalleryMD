# GalleryMD: Deno Edition

**Project Type:** Web-based Markdown Gallery

**Overview:**
GalleryMD is a modern, fast, and secure web application for browsing Markdown-based case studies. Built with Deno and designed for Docker deployment, it features instant loading, background scanning, and a beautiful light-themed UI.

## Features
- Scans folders for `CASESTUDY.md` and associated images
- Displays case studies in a responsive grid
- Fast cache-based loading with background refresh
- Manual refresh and always-visible loading spinner
- Secure Deno backend with Oak
- Runs in Docker for easy deployment

## Technologies
- Deno (Oak framework)
- HTML5, CSS3 (light theme), JavaScript
- Docker (Alpine-based)

## Usage
1. Build Docker image: `docker build -t gallerymd-deno .`
2. Run: `docker run -p 3000:3000 -v /your/docs:/docs -e MD_ROOT=/docs gallerymd-deno`
3. Browse at `http://localhost:3000`
